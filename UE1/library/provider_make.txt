option 2: when passing PROVIDER_NAME via command line
g++ -std=c++14 -D PROVIDER_API_EXPORTS -D PROVIDER_NAME="\"test\"" -o provider.o -I./ -c ./provider.cpp

1.2) linking:
g++ -std=c++14 -dynamiclib -o provider ./provider.o -L./
